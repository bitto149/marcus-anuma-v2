
# Marcus Parte 100 - Terminal Interface

Questo modulo fornisce un'interfaccia terminale per inviare comandi a Marcus e registrare automaticamente tutto nella cartella marcus_memories/.

## File inclusi
- marcus_terminal_interface.js: Terminale interattivo con prompt Marcus, salvataggio memoria, cronologia.
- README_100.txt: Questo file.

## Uso
1. Assicurati di avere una cartella marcus_memories/ nella directory principale.
2. Esegui: node marcus_terminal_interface.js
3. Scrivi comandi, ricevi risposta simulata, tutto viene salvato.
